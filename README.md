# ⚔️ FitWarrior AI — PWA

> Train Smart. Eat Right. Level Up Daily.

## 🚀 Deploy to GitHub Pages (FREE — 5 minutes)

### Step 1: GitHub pe nayi repo banao
1. github.com pe jao → **New Repository**
2. Name: `fitwarrior-ai`
3. Public rakho ✓
4. **Create repository** click karo

### Step 2: Files upload karo
1. **"uploading an existing file"** click karo
2. Ye 4 files drag karo:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icons/` folder (icon-192.png + icon-512.png)
3. **Commit changes** click karo

### Step 3: GitHub Pages ON karo
1. Repo → **Settings** tab
2. Left sidebar → **Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** → **/ (root)**
5. **Save** click karo

### Step 4: URL milega
```
https://YOUR_USERNAME.github.io/fitwarrior-ai/
```
2-3 minutes me live ho jayega!
